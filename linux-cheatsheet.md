# Linux — ściąga komend

## Nawigacja
pwd       - pokazuje, gdzie jestem
ls        - lista plików w folderze
ls -la    - lista z ukrytymi plikami i szczegółami
cd nazwa  - wejście do folderu
cd ..     - folder wyżej
cd ~      - powrót do domu

## Pliki i foldery
mkdir nazwa   - tworzy folder
touch plik    - tworzy pusty plik
cat plik      - pokazuje zawartość pliku
rm plik       - kasuje plik (bez kosza, trwale!)
